# Отладка проблемы с открытием письма

## Проверьте API напрямую:

```bash
# Замените ID на ваш
curl http://localhost:8080/api/v1/mailbox/ac1339f0-a773-4353-b4fa-298b6f2084cc/messages/71a6d534-9be2-493a-a60a-e541790ca508
```

Должен вернуть JSON с полями:
- `body_text` - текстовое содержимое
- `body_html` - HTML содержимое (может быть пустым)

## Проверьте в браузере:

1. Откройте консоль разработчика (F12)
2. Перейдите на страницу письма
3. Проверьте:
   - Ошибки в консоли
   - Сетевые запросы (Network tab)
   - Что возвращает API запрос

## Возможные проблемы:

1. **Письмо не содержит body_text и body_html** - проверьте, что письмо было правильно сохранено
2. **Ошибка CORS** - проверьте настройки CORS на сервере
3. **Неправильный URL** - проверьте, что ID письма правильный

## Проверьте базу данных:

```bash
docker exec -it tempmail-postgres psql -U postgres -d tempmail -c "SELECT id, subject, body_text, body_html FROM messages WHERE id = '71a6d534-9be2-493a-a60a-e541790ca508';"
```
