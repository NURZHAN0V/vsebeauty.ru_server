# Настройка для получения писем из интернета

## ✅ MX запись настроена:
```
vsebeauty.ru → mail.vsebeauty.ru
```

## Шаги настройки:

### 1. Проверьте A запись:
```bash
dig A mail.vsebeauty.ru
```
Должна указывать на IP вашего сервера.

### 2. Изменения в docker-compose.yml уже сделаны:
- SMTP_PORT=25
- Порт 25:25 проброшен

### 3. Откройте порт 25 в файрволе:
```bash
# Проверьте файрвол
ufw status

# Откройте порт 25
ufw allow 25/tcp

# Или для iptables
iptables -A INPUT -p tcp --dport 25 -j ACCEPT
iptables-save
```

### 4. Перезапустите контейнер:
```bash
docker compose down
docker compose up -d
```

### 5. Проверьте порт 25:
```bash
netstat -tuln | grep ":25 "
# Должно показать: tcp  0  0  0.0.0.0:25  LISTEN
```

### 6. Проверьте логи:
```bash
docker logs tempmail-app | grep -i smtp
# Должно быть: "SMTP-сервер запущен на порту 25"
```

## ⚠️ Важно:

- Порт 25 требует root прав (Docker может запускать)
- Многие хостинги/провайдеры блокируют порт 25
- Если порт заблокирован провайдером - письма не дойдут

## После настройки:

Отправьте тестовое письмо с Яндекс/Gmail на ваш временный email. Письмо должно появиться в логах и базе данных.
